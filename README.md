# mycfg
## Git Bare Repository
This repository is intended to be pulled as a bare repositiry as described by [this article.](https://mjones44.medium.com/storing-dotfiles-in-a-git-repository-53f765c0005d) This respository is installed on the initial
machine in the *$HOME/.mycfg* directory and has a zsh alias named *config* to work with it. You can see the *config* alias in the *.zshrc* file.
